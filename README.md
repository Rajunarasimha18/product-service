Product Management Service for the E-Kart Microservices Architecture built using Spring Boot.

🚀 Tech Stack
Java 21
Spring Boot 3.5.7
Spring Data JPA
MySQL
Maven
📌 Overview

This service handles product-related operations such as product management, categories, pricing, inventory, and product APIs within the microservices architecture.

⚙️ Features
Product Management
Inventory Handling
REST APIs
Database Integration
